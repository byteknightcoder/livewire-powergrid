## 1. Requirements

- [Laravel 8x](https://laravel.com/docs/8.x/installation)

- [Tailwind](https://tailwindcss.com/docs/guides/laravel) or [Bootstrap 5](https://getbootstrap.com/docs/5.0/getting-started/introduction/)
